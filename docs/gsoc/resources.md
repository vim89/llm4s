# Student Resources

Interested in contributing to LLM4S or applying for future GSoC terms? Here is everything you need to get started.

## How to Apply
We participate through the **Scala Center**. When applications open:
1.  Check the [Ideas Page](https://github.com/llm4s/llm4s/blob/main/Google%20Summer%20of%20Code/Project%20Ideas/2026.md).
2.  Join our Discord to discuss ideas with mentors.
3.  Submit a proposal focusing on Type Safety, Functional Programming, and AI innovation.

## Onboarding for Contributors
If you have been selected or want to start contributing now:

1.  **Join the Community**:
    - [LLM4S Discord](https://discord.gg/4uvTPn6qww) - The primary hub for discussion.
    - [GitHub Issues](https://github.com/llm4s/llm4s/issues) - Find "good first issues".

2.  **Dev Environment Setup**:
    - Install JDK 21+ and SBT.
    - Run `sbt test` to ensure the core builds correctly.
    - Check out the [Contribution Guide](/contributing).
